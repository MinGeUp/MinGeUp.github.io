java

主要特点：

面向对象：Java采用面向对象编程（OOP）范式，通过类和对象实现封装、继承和多态等特性。
平台无关：Java代码编译成字节码，可以在任何支持Java虚拟机（JVM）的设备上运行。
垃圾回收：Java具有自动内存管理和垃圾回收机制，减轻了程序员的负担。
巨大的类库：Java提供了丰富的类库，涵盖了各种应用开发需求。
多线程支持：Java内置了对多线程的支持，简化了并发编程。

概括

关键字 意思 备注，常用
public 公有的 可跨包
protected 受保护的 当前包内可用
private 私有的 当前类可用
class 类 public class A(){} 有已实现方法体，类名需要与文件名相同
interface 接口 public interface B(){} 有方法体，但没有实现
abstract 声明抽象 public abstract class C(){} 可以有也可以没有已经实现的方法体
implements 实现 用于类或接口实现接口public class A implements B(){}
extends 继承 用于类继承类 public class A extends D(){}
new 创建新对象 A a=new A(); A表示一个类
import 引入包的关键字 当使用某个包的一些类时，需引入包路径
package 定义包的关键字 当前类的包路径
byte 字节型 8bit
char 字符型 16bit
boolean 布尔型 –
short 短整型 16bit
int 整型 32bit
float 浮点型 32bit
long 长整型 64bit
double 双精度 64bit
void 无返回 public void A(){}
true 真
false 假
if 如果 if(){} 括号true则执行
else 否则，或者 常与if连用，用法相同
while 当什么的时候 while () {} 括号true则执行
for 满足三个条件时 for ( ; ; ){} 循环判断
switch 开关 switch(表达式) {} 与case互用
case 返回开关里的结果 case 常量:语句;
default 默认
do 运行 do {}while ();
break 跳出循环
continue 继续 中断本次循环，并并开始下一次
return 返回 return 一个返回值类型
instanceof 实例 测试它左边的对象是否是它右边的类的实例，返回boolean
static 静态的 修饰属性和方法，调用直接使用类名
final 最终的 修饰方法、属性和类，类不能被继承，方法不能被重写，属性是常量
super 调用父类的方法 代表父类对象
this 当前类 代表当前类对象
synchronized 线程,同步 对象锁，可修饰类、方法和代码块，多线程下只能一个线程执行
catch 处理异常 与try互用，有异常抛出，则转到catch块去处理
try 捕获异常 与catch互用
finally 最后执行 一定会执行
throw 抛出一个异常 主动抛出错误信息
throws 方法异常可能被抛出 把异常交给他的上级管理，自己不进行异常处理
enum 枚举，列举型别
assert 断言
